# CRM Laravel

مشروع CRM بسيط مبني باستخدام Laravel و PHP و MySQL عبر XAMPP.

يدعم المشروع سجل ملاحظات مستقل لكل عميل داخل صفحة التفاصيل، بجانب المتابعات المجدولة.

بيانات المدير الافتراضي:

```text
البريد: admin@crm.local
كلمة المرور: admin12345
```

## التشغيل

1. شغل Apache و MySQL من XAMPP.
2. افتح PowerShell داخل مجلد المشروع:

```powershell
cd "C:\Users\alhay\Documents\Codex\2026-06-08\crm-by-larvel-php-sql\crm"
php artisan serve
```

3. افتح الرابط:

```text
http://127.0.0.1:8000/dashboard
```

## قاعدة البيانات

اسم قاعدة البيانات:

```text
crm_laravel
```

تحديث الجداول عند الحاجة:

```powershell
php artisan migrate
```

## الشركات

يدعم النظام شركتين:

- بيت المصور: نشاط تعليمي في مجال التصوير الفوتوغرافي، ويتابع المهتمين بالدورات القادمين من إعلانات التواصل وجوجل حتى يصبحوا مشتركين حقيقيين.
- فيدا برودكشن: نشاط ميديا برودكشن وتصوير منتجات وصناعة محتوى مرئي، ويتابع العملاء المحتملين حتى يصبحوا عملاء حقيقيين.

حالات المتابعة الحالية:

- عميل محتمل
- تم التواصل
- قيد المتابعة
- مشترك / عميل حقيقي
- مغلق / غير نشط

دورات بيت المصور الحالية:

- اساسيات التصوير
- احتراف الاضاءة
- احتراف تصوير الاعراس
- احتراف تصوير الاطعمة

يمكن تعديل هذه الدورات أو إضافة دورات وخدمات جديدة من شاشة الدورات والخدمات داخل النظام.

## استيراد العملاء من Google Sheet

1. افتح شاشة الاستيراد من داخل النظام.
2. يمكن تحميل قالب CSV جاهز من زر تحميل القالب.
3. من Google Sheet اختر تنزيل الملف كـ CSV.
4. اختر الشركة الافتراضية للصفوف التي لا تحتوي على شركة.
5. ارفع ملف CSV.

الأعمدة المقترحة:

- اسم العميل
- رقم الجوال
- البريد الإلكتروني
- الشركة التابعة لنا
- شركة العميل
- المصدر
- الدورة المطلوبة
- الخدمة المطلوبة
- الحالة
- ملاحظات
- مسؤول المتابعة

النظام يحاول تحديث العميل الموجود إذا وجد نفس البريد الإلكتروني أو رقم الجوال، وإلا يضيف العميل كسجل جديد.
